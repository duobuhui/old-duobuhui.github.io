违纪学生行政处罚规定

为严肃学校纪律，确保良好的校园学习和生活秩序，特制定本规定。
对于极少数严重违反校纪校规或多次犯错经教育不改的学生，学校视情节将给予批评教育和行政纪律处分。处分等级分为：警告、严重警告、记过、留校察看、开除学籍。

学生违纪处分细则

有下列违纪行为，学校将根据学生违纪情节、造成的后果和认错态度给予相应的行政处分

  1. 一学期内无故旷课达4节（迟到或早退三次折算一节旷课，每次迟到或早退15分钟以上的视为旷课一节，旷课一天的按8节计）取消评优资格；旷课达10节可给予警告处分;旷课在20—30节的，给予严重警告处分；旷课达31节或连续旷课三天的，给予记过处分；旷课达41节或连续旷课一周的，给予留校察看处分；旷课累计达80课时以上或连续旷课二周的，开除学籍。
  2. 上课打闹等扰乱课堂纪律，不服从任课教师管理，致使教师无法正常上课的；不服从教育，谩骂、威胁、殴打教职工的；一学期无故不交作业累计10次以上的，或连续两周有一科以上作业无故不交的，视情节轻重，给予警告或严重警告处分。
  3. 期中、期末考试作弊，该学科成绩以零分记，并给予警告或严重警告处分，详见《学生守则及违纪处理规定》。
  4. 私自出校门或翻墙外出的；故意损坏公物的（赔偿另按有关规定执行），视情节轻重，给予警告或严重警告处分。
  5. 男女学生行为过于亲密，造成不良影响或后果的；欺侮同学、聚众打架的；私自窃取他人钱物的，视情节轻重，给予警告、严重警告或记过处分。
  6. 受到处分后又违纪，再次应受纪律处分的，应加重处罚。
  7. 有其他违纪行为，参照以上细则给予适当处分。

学生处分与撤销

  1. 学生违纪后，班主任或年级组长找当事学生个别谈话，弄清违纪事实并给予教育，责令学生写出书面检查，并提出处理意见报德育处。
  2. 给予学生行政处分的，由德育处审定，报学校行政会议批准公示；给予留校察看或开除学籍的，要按规定上报有关部门备案。
  3. 学生受处分前，班主任要将该生家长通知到学校，向家长汇报该生违纪事实及处分原因。
  4. 学生受处分后，认错态度好，确有悔改表现的，在受警告或严重警告处分一学期后，受记过、留校察看处分一学年后，由学生本人提出申请，班主任在征求各方意见后，提出处分撤销意见，并报德育处审批公示。
  5. 学生处分决定等有关材料存入学校文书档案和学生档案。处分撤销后，将有关材料从学生档案中取出。

宿舍处罚条例

为强化宿舍纪律管理，培养全体寄宿生养成良好的寄宿习惯。将寄宿学生违纪现象根据情节轻重分为三个类别，分别是一类违纪、二类违纪、三类违纪。
违纪一旦发生，德育处会同班主任应及时按照相应规定进行处罚。其中，“一类违纪”的处罚为：从违纪现象查实的第二天起勒令退宿，并向全校通报批评；“二类违纪”的处罚为：首次违纪走读一周（5天），如同一学期内“二类违纪”再次发生，则视作“一类违纪”进行相应处罚；“三类违纪”的处罚为：首次违纪交由班主任进行批评教育，如同一学期内三类违纪再次发生，则视作“首次二类违纪”进行相应处罚，若同一学期内三类违纪第三次发生，则视作“一类违纪”，进行相应处罚。
对于违纪的学生，德育处将根据情节严重程度决定是否取消该学期各类评优资格直至给予行政处分。

一类违纪

  1. 顶撞、谩骂舍管和值班老师；
  2. 夜不归宿；未经舍管人员同意私自留宿非本宿舍人员；合床住宿；擅闯异性宿舍；未经许可擅自留在宿舍内不上课；
  3. 抽烟、喝酒、赌博、盗窃、打架斗殴；
  4. 攀爬宿舍围栏和窗户；
  5. 违规用电如私拉电线、使用拖线板、小家电；违规烧废纸、点蜡烛、蚊香、爆竹
  6. 拨弄公共配电箱、摄像头；故意损坏公共设施设备及家具等；
  7. 传阅不健康书画视频；
  8. 22: 30至次日6: 00在宿舍或走廊大声暄哗、怪叫、起哄、打闹，乱串宿舍（含周5晚至周六晨）；
  9. 其它经德育处认定为一类违纪的严重违纪情节。

二类违纪

  1. 打牌、玩游戏、使用手机、平板等电子产品；
  2. 向楼道或窗户外泼水，乱扔垃圾等；
  3. 22: 00熄灯前乱窜宿舍、大声暄哗、怪叫起哄、打闹嬉戏;
  4. 22: 00熄灯后在大声讲话，影响他人休息；
  5. 熄灯后在宿舍使用强光手电等照明设备；
  6. 私自调换宿舍、床位等
  7. 在宿舍内聚餐等
  8. 22: 00熄灯后仍在串宿舍、楼层，
  9. 其它经德育处认定为二类违纪的严重违纪情节。

三类违纪

  1. 不遵守宿舍大门关启制度，晚上21: 50后进入宿舍的和上课期间早上6: 30后离开宿舍的；
  2. 上课期间，晚自修结束后在操场上打球；
  3. 周六在操场上活动，经保安督促仍不能按时离场；
  4. 接到卫生整改通知后，未在指定时间回宿舍进行整改的。
  5. 22: 00熄灯以后，未及时关闭宿舍及厕所照明。
  6. 22: 00熄灯后讲话，经提醒仍不改正；
  7. 同一违规行为一月内累计满三次的；
  8. 其它经德育处认定为三类违纪的严重违纪情节。

夜自修处罚条例

一、参加晚自修的学生有违纪现象发生，根据情节轻重分为三个类别。分别是一类违纪、二类违纪、三类违纪。
违纪一旦发生，将由当班教师和领导按照相应规定进行处罚。其中，“一类违纪”的处罚为：从违纪现象查实的第二天起勒令退宿，并向全校通报批评；“二类违纪”的处罚为：首次违纪走读一周（5天），如同一学期内“二类违纪”再次发生，则视作“一类违纪”进行相应处罚；“三类违纪”的处罚为：首次违纪交由班主任进行批评教育，同一学期内三类违纪再次发生，则视作“首次二类违纪”进行相应处罚，若同一学期内发生第三次“三类违纪”，则视作“一类违纪”，进行相应处罚。对于违纪的学生，德育处将根据情节轻重决定是否取消该学期各类评优资格直至给予行政处分。

一类违纪

  1. 大声暄哗、打闹、嬉戏；
  2. 使用自带的手机、MP3、MP4及其他电子设备；
  3. 抄袭作业；
  4. 其他经德育处认定为一类违纪的破坏晚自习秩序或不认真学习行为。

二类违纪

  1. 无故缺席；
  2. 玩魔方、打牌、下棋等；
  3. 晚自习结束后继续在户外从事体育活动；
  4. 随便讲话、传纸条聊天；
  5. 在宿舍以外的地方逗留，9: 50前未回到宿舍；
  6. 其他经德育处认定为二类内外秩序或不专心学习的行为。

三类违纪

  1. 迟到；
  2. 吃东西、喝饮料；
  3. 课桌内学习用品杂乱摆放；
  4. 打瞌睡；
  5. 看闲书(原则上指“非本校图书馆借阅”或“非教师推荐”的图书、杂志等）；
  6. 衣冠不整（如穿拖鞋、着装不得体等）参加自习；
  7. 课上经常性借学习用品、材料；
  8. 其他经德育处认定为三类违纪的影响课内外秩序或学习不自觉的行为；
