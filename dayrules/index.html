20年新高二化学

一、上学放学要求

  1. 学生按时到校、离校，不迟到早退，因病因事不能按时到校者必须向班主任请假；进出校园主动出示“胸卡”，在校期间要自觉佩戴“胸卡”；放学静堂后，走读生应马上离校回家，寄宿生及时处理个人生活。
  2. 进出学校主动向值勤老师及同学问好示意。
  3. 上学时要带齐学习用品，凡是对正常教育教学秩序有妨碍的物品一律禁止带入校内，如游戏机、棋牌、品位低的书刊、宠物、化妆用具、移动通讯工具、MP3、MP4、影碟、滑板、链条、打火机、香烟、酒及含酒精的饮料、易爆易燃物、腐蚀类物品、玩具枪、刀具等。凡违反规定私自带入校园的，除没收所带物品外，还要对当事人进行教育、处理。
  4. 进入学校必须穿戴整洁，朴素大方。
  5. 凡经学校批准同意骑电瓶车、自行车的同学，进出门应下车推行。电瓶车、自行车由学校统一标记，在指定地点整齐停放，以备检查。
  6. 到校后将家庭作业交科代表，自觉参加早自习和做好早、晚卫生值曰工怍。
  7. 离校时发现寻衅滋事者或可疑迹象，应主动回校报告老师。
  8. 放学路上遵守交通规则，注意交通安全。

二、学习常规

（一）早读要求

  1. 严格按学校规定的时间进行早读。
  2. 严格服从老师及班干部安做到“一有五到”。“一有”指有明确的早读任务，“五到”指书到、手到、眼到、口到、心到。

（二）课前要求

  1. 用品准备：课前，要准备好上课所需的有关学习用品，书本、笔记本、文具盒一律整齐地排放在课桌的两角，其他与学习无关的用品，包括水杯等均需放入桌内。上课时不乱翻桌内东西，以免影响听课效果。
  2. 精力准备：平时要积极参加体育锻炼，合理地安排好作息时间，保持旺盛的精力。课前不做剧烈的运动，以良好的精神状态去听课。
  3. 心理准备：课前两分钟准备音乐响起，必须迅速、安静、有序地进入教室，安静坐好，等候老师上课；上课时要保持稳定良好的心理状态，不凭兴趣去听课。
  4. 知识准备（预习准备）：课前按老师要求提前预习新课，带着问题有目的地、认真地去听课。
    ① 预习要及时。根据教师的教学进度和教材的难易程度，适当地提前预习新课。
    ② 预习态度要认真。认真细致阅读教材，动脑分析，不要粗枝大叶、马马虎虎看一遍了事。
    ③ 预习要讲求质量。基本了解教材内容，把握教材的结构，尽力弄懂教材中的新概念；找出重点和难点；明确新课要解决的问题和预习中不懂的问题，为上新课做好准备。
    ④ 预习要做好笔记。记好预习重点、难点问题以及预习中尚未解决的问题。

（三）上课要求

  1. 上课起立快、静、齐。上课铃响后，老师宣布“上课”，班长喊“起立、敬礼（行鞠躬礼）”，老师回礼、检查认可后宣布“请坐下”，学生方可坐下。
  2. 铃响后不随意离开教室。因故迟到者，应在教室门口向老师喊”报告〃，经教师同意后方能进入教室落座。
  3. 遵守课堂纪律，精神饱满，保持安静，坐姿端正，专心听讲，积极思维，适当笔记，不做小动作，不打瞌睡，不喝水，不吃零食，不看与本科无关的书籍，努力掌握老师的讲课内容。
  4. 用于提问、敢于发问，敢于发表自己的见解。回答老师提问或向老师发问时，要先举手，得到老师同意后，起立发言，身体站直，声音洪亮，经老师同意后坐下。
  5. 实验课要明确目的要求，仔细观察老师示范动作，按规定的操作步骤进行观察分析，保证实验室安全，写好实验报告，爱护实验器材，节省实验材料，实验课结束，清洗、整理、存放好实验物品；若有损坏及时报告并按要求赔偿。
  6. 上体育课，应换穿运动服、运动鞋并认真做好准备活动，身体不适及时向老师请假，活动时注意安全和自我保护。
  7. 遇来宾参观或听课，应有礼貌接待，下课后应待来宾离开教室后，方可离开座位。
  8. 学生不得在教室内进行任何有碍教学的活动。
  9. 下课时老师宣布“下课”，班长喊“起立、敬礼（行鞠躬礼），老师回礼后，学生解散下课，下课后应请老师先行。

（四）自习要求

  1. 自修课不迟到、早退，不外出，有事请假，无故不到者按旷课处理。
  2. 自修课应专心致志地学习，不回头，不随便离开或更换座位，不看与学习无关的书刊、报纸，保持教室的安静，不打扰他人学习和工作。
  3. 寄宿生夜自修要求：定人、定点、定时自习；自修期间，不玩、不闹、不说话、不回头，不听耳机、不看闲书及报刊杂志，不喝水、不吃零食，提高学习效率，请教问题应在课间进行；待用的书本整齐放在桌面两角；坐姿端正，不东倒西歪；着装整洁，不得穿睡衣及拖鞋上课；保持自修教室卫生整洁，离开前关门、窗、空调、灯；夜自习结束后，及时从主干道回宿舍，不在非宿舍区逗留。

（五）复习要求

复习是学生自己或在教师指导下，巩固所学的知识，加深对知识的理解，为继续学习打好基础的重要过程。复习包括课后复习、单元复习、阶段复习和期末复习等。复习的基本要求是:

  1. 及时复习。课后复习要当天完成，以巩固当天所学的知识。在教师讲完一个单元、一个阶段的知识或讲完全部教材后，要及吋复习，及时巩固知识。
  2. 复习要有针对性，要抓住要点，对一些重要的基本概念和基础知识，通过理解加深记忆。
  3. 复习要突破难点，解决自己在学习中的疑难问题。针对自己学习中的薄弱环节，精心弥补“漏洞”和欠缺。
  4. 复习要注意学习归纳总结的方法，使知识条理化、系统化。
  5. 记好复习笔记。通过复习，把学科的重点和要点记在笔记本上，既便于查阅又可使所学的知识形成系统。

（六）作业要求

  1. 作业习惯好：先复习，后作业
  2. 作业态度端正：独立思考，格式规范，字迹工整，条理清晰，正确使用标点符号；除作图外，各科作业均不得使用铅笔。认真独立完成作业，不懂就问，不拖拉、抄袭、拒交作业；老师讲评后，及时订正。
  3. 解题科学：总结答题规律，答题要有一定的速度；严格按照学科老师要求的格式规定，认真总结解题规律。
  4. 作业整洁：不在作业本上乱涂乱画，不随意撕扯作业本。
  5. 作业完整：认真完成听音、背诵、阅读、思考等非书面作业。做好复习和预习工作。积极参加各种研究性学习和课题研究活动。

（七）考试要求

考试是学生平时学习的总结，是检查学生知识掌握程度、评价学生学习效果的一种有效手段，也是学生学习过程中的一个重要环节。学生考试的基本要
求是：

  1. 要正确对待考试。树立正确的考试观，重视考试成绩又不要过分看重，胜不骄，败不馁；要注意调节自己的心理和情绪。
  2. 要遵守考场纪律，杜绝作弊。违纪处理详见《考生守则及违纪处理规定》
  3. 要注意卷面整洁。答题时不要随想随写，随写随改；字迹不要潦草；不要在试卷上乱勾乱划，要注意保持试卷的整洁。
  4. 要学会写试卷分析，养成分析、总结的习惯。

三、课间休息

  1. 下课后，先准备下节课所需的学习用品，方可休息。值日生檫净黑板、黑板槽、讲台，保持讲台上物品整洁。
  2. 遵守午间静休纪律。课间和午间合理作业休息，不影响他人。
  3. 课间可出教室作远望或其他轻微活动，不高声暄哗，不追逐打闹，不吹哨起哄，不攀爬护栏，不开展剧烈运动。
  4. 课间遇到老师或来宾，要主动问好，指路，并注意礼让。
  5. 课间未经班主任许可，不得随意进出校门。不接待亲朋好友探访。
  6. 上下楼梯或在走道内行走要靠右侧。

四、课外活动

  1. 积极参加学校组织的文娱、体育、科技和各类活动。
  2. 认真参加广播操，因故不能参加者，须提前请假，征得同意。广播操集队要快、静、齐，做操要整齐，精神饱满，动作准确到位。
  3. 认真做好眼保健操，注意读写姿势和用眼卫生。
  4. 积极参加课外体育活动，先整队，后活动，服从安排，遵守纪律，爱护运动场地、器材，注意运动安全。不得在室内逗留，因故不能参加者，须履行请假手续。
  5. 不在非运动场所开展篮球、排球、足球等运动。

五、集会纪律

  1. 每周一及学校重要活动日，全天穿校服。
  2. 参加集会、观看演出和比赛不无故缺席，不迟到早退，请假须有书面假条。
  3. 集会前，整队清点人数，排队准时进场，按指定位置就坐。
  4. 开会时，坐（站）姿端正，专心听讲，不交头接耳，不看书报，不做作业，不做动作，不吃东西，不乱扔垃圾，不喝倒彩，保持会场文明、安静、有序。
  5. 集会议程中需唱国歌，要求歌声响亮；学生大会发言要先向师长和听众致礼，发言结束应道谢；领奖要动作迅速、态度恭敬，双手接奖品，并鞠躬；文明观看演出、比赛，适时、适度鼓掌致意；重要报告要携带笔记本认真做好笔记。
  6. 比赛活动中，与对手保持友好的态度；服从评委裁决（若对学生评委的能力有异议，可在比赛前或赛后通过正当程序提出）
  7. 会议期间不随意离开；散会时在主持人的指挥下，安静、有序退场。
  8. 参加升旗仪式，要统一穿校服（含校裤）并拉好拉链，态度认真、严肃，唱《国歌》要响亮，奏国歌时要肃立、脱帽、行注目礼；认真倾听国旗下讲话。

六、卫生保健

（一）卫生环卫工作

  1. 自觉维护学校室内外环境卫生。环境卫生实行分班保洁制，各班排好卫生值日表，坚持保洁区每天两小扫，每周一大扫，学期开始、结束和节假日前后例行大扫除，保持校园环境持久清洁。做到“四定”：定人、定时间、定地点、定措施。
  2. 保洁区要求做到“五无”（无痰迹、无纸屑、无乱砖垃圾、无杂草杂木、无积水）。室内卫生要求做到“洁、齐、美”（窗明几净、地面清洁，用具摆好、桌椅整齐，布置大方、环境优美）。
  3. 宿舍要保持室内外、窗、地面干净，墙壁无尘、四角无蛛网，被褥、家具、毛巾考用具摆放整齐。
  4. 动时彼此团结友爱，发扬协作精神，注意安全，禁止爬阳台擦洗窗户外侧，杜绝事故发生。
  5. 坚持每周检查评比，对集体和个人的保洁状况予以评价并开展表扬与批评。
  6. 有传染病即使报告校医，及时采取隔离等措施。

（二）个人卫生制度

  1. 个人卫生要求做到“六勤”“五不”。
    六勤：勤洗手、勤剪指甲，勤理发，勤洗澡，勤换衣服，勤刷牙。
    五不：不随地吐痰和口香糖残渣；不随地丟纸屑果壳杂物；不乱倒、乱丟垃圾，不从楼上向下抛洒物品，楼道垃圾桶盖开启后及时盖好；不吃生水及不洁食物；不乱涂乱画、污损墙壁。
  2. 认真做课间操和眼保健操。

七、爱护公物

  1. 爱护校园公物：不践踏草坪，不采摘花果，不污染池塘，不损坏体育和校园内公共设施。
  2. 爱护室内公物：不在课桌、黑板、布告栏、门窗、地面、围栏、墙壁上乱涂乱刻，维护保管好劳动工具、灯具、电视、广播、空调、风扇、饮水等设施设备，遵守微机室、多媒体室、阶梯教室、实验室管理规定，爱护微机、电教和实验设备，维护学校网络安全。门窗及时启闭，随时固定好，放学、夜自修结束或离开宿舍时必须把门窗全部关好。
  3. 遵守学校图书和阅览室管理规定：爱护图书，不弄脏、折角、撕页，阅览室的报刊杂志不擅自带出室外，借阅书刊要按时归还，图书损坏遗失要赔偿。
  4. 体育课和课外活动，运动器具由体育委员负责借用和及时归还。
  5. 借用劳动工具和其它公物，要经管理人员同意，使用后及时归还，如有丟失或损坏应赔偿。
  6. 节约水、电，不开无人灯，用完水后立刻关闭水龙头；发现公共区域漏水或设施损坏及时上报总务部门。

八、用餐要求

  1. 在规定时间到食堂用餐。
  2. 有序进入食堂，不奔跑、不争先恐后。
  3. 进入食堂后，服从值勤管理，有序排队，不插队或变相插队。
  4. 除非凭班主任开具的证明代买病号饭外，否则不允许帮别人代买饭菜，不将饭菜带出食堂；教学区及公共设施区域不得吃方便面。
  5. 珍惜粮食，吃剩饭菜倒入指定的泔脚桶中；餐毕，餐具有序摆放在指定收集箱内，保持餐桌椅整洁。
  6. 文明用餐，保持安静，尊重食堂工作人员，不随便出入厨房。
